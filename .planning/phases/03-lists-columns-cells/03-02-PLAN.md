**C) Add require_player() to src/auth/session.php:**

Append after the existing `require_coach()` function:

```php
/**
 * Require a player session. Per D-12:
 * Checks $_SESSION['role'] === 'player', sets RLS team context, redirects on failure.
 */
function require_player(): void {
    check_session_timeout();
    if (empty($_SESSION['user_id']) || ($_SESSION['role'] ?? '') !== 'player') {
        redirect('/login');
    }
    $pdo = get_db();
    reset_rls_context($pdo);
    set_team_context($pdo, (int)$_SESSION['team_id']);
}
```
