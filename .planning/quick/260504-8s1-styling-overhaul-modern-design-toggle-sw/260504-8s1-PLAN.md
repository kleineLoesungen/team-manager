In the table body cell rendering block (around line 124-129), replace the bare checkbox for `boolean` columns with a Bootstrap 5 form-switch:

Current:
```php
if ($col['data_type'] === 'boolean') {
    $checked = ($val === '1') ? 'checked' : '';
    echo '<input type="checkbox" class="form-check-input"
          name="cells[' . (int)$player['id'] . '][' . (int)$col['id'] . ']"
          value="1" ' . $checked . '>';
```

Replace with:
```php
if ($col['data_type'] === 'boolean') {
    $checked = ($val === '1') ? 'checked' : '';
    echo '<div class="form-check form-switch mb-0">'
        . '<input class="form-check-input" type="checkbox" role="switch"'
        . ' name="cells[' . (int)$player['id'] . '][' . (int)$col['id'] . ']"'
        . ' value="1" ' . $checked . '>'
        . '</div>';
```

This wraps the checkbox in Bootstrap's `.form-check.form-switch` div which renders a pill-shaped toggle. The `role="switch"` attribute provides accessibility semantics. The `mb-0` removes default bottom margin so the toggle sits flush in the table cell. No change to the name/value/checked logic — form submission behavior is identical.

Also add a small CSS rule to the existing `<style>` block in `render_layout_head()` (already edited in Task 2) to make the switch touch-friendly — the Bootstrap default is 2rem wide; at small table-sm size this is already good. No additional CSS needed beyond what Bootstrap 5 provides for `.form-switch`.
