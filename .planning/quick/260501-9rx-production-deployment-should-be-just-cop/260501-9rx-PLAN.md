# Quick Plan 260501-9rx: Production Deployment Script

## Goal

Deployment = one command: `./deploy.sh user@server:/var/www/team-manager`. A `deploy.sh` rsync script copies the app files (excluding dev artifacts). A `public/.htaccess` enables Apache URL rewriting without manual server config.

## Tasks

### Task 1: deploy.sh — rsync deployment script

**Files:**
- `deploy.sh` — new: rsync wrapper for deploying to a webserver

**Action:**

Create `deploy.sh` at project root. The script:
- Takes `DESTINATION` as first argument (e.g. `user@server:/var/www/app`)
- Uses `rsync -avz --delete` to copy all app files
- Excludes: `.git/`, `.planning/`, `docker/`, `docker-compose.yml`, `.env`, `.env.docker`, `.env.example`, `deploy.sh`
- Prints clear before/after output
- Exits with error if no destination given
- chmod 755

Content:
```bash
#!/usr/bin/env bash
set -euo pipefail

DEST="${1:-}"

if [[ -z "$DEST" ]]; then
    echo "Usage: ./deploy.sh user@server:/var/www/team-manager"
    exit 1
fi

echo "Deploying to $DEST ..."

rsync -avz --delete \
    --exclude='.git/' \
    --exclude='.planning/' \
    --exclude='docker/' \
    --exclude='docker-compose.yml' \
    --exclude='.env' \
    --exclude='.env.docker' \
    --exclude='.env.example' \
    --exclude='deploy.sh' \
    . "$DEST"

echo ""
echo "Done. Don't forget:"
echo "  1. Set environment variables on the server (see README.md)"
echo "  2. Point webroot to public/"
echo "  3. On first deploy: run database/schema.sql and database/rls_policies.sql"
```

**Verify:** `deploy.sh` exists at root, is executable, has correct rsync excludes.

**Done:** `feat(deploy): add deploy.sh rsync script for webserver deployment`

### Task 2: public/.htaccess — Apache URL rewriting

**Files:**
- `public/.htaccess` — new: Apache rewrite rules for front controller

**Action:**

Create `public/.htaccess` with:
```apache
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^ index.php [QSA,L]
```

This enables Apache deployments without modifying VirtualHost config, as long as `AllowOverride All` is set.

**Verify:** `public/.htaccess` exists with RewriteEngine rules.

**Done:** `feat(deploy): add public/.htaccess for Apache URL rewriting`
