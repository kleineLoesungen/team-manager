# Quick Task 260503-06c: Add Delete List for Coaches (Two-Step Confirmation)

**One-liner:** Two-step POST-only list deletion with danger zone card in settings, team_id ownership guard, and DB cascade cleanup.

## What Was Built

Coaches can now delete a list from its settings page via a two-step confirmation flow:

1. The settings page gains a "Gefahrenzone" danger card at the bottom with a "Liste löschen" button.
2. Clicking the button POSTs `confirm=0` to `/coach/lists/{id}/delete`, which renders a confirmation page naming the list and warning that deletion is irreversible.
3. Confirming sends `confirm=1`, which executes `DELETE FROM lists WHERE id = ? AND team_id = ?` and redirects to `/coach/lists`. PostgreSQL cascades remove all associated columns, cells, and list_global_columns rows automatically.
4. Cancelling returns to `/coach/lists/{id}/settings`.
