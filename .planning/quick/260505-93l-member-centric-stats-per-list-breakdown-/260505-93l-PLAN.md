**Query A — lists with global columns for this member (player sees public + protected only):**
```sql
SELECT DISTINCT l.id, l.name, l.date
FROM lists l
JOIN list_columns lc ON lc.list_id = l.id
JOIN columns c ON c.id = lc.column_id
    AND c.team_id = :team_id AND c.list_id IS NULL AND c.is_active = TRUE
WHERE l.team_id = :team_id
  AND l.visibility IN ('public', 'protected')
ORDER BY l.date DESC NULLS LAST, l.name
```

Bind `[$team_id, $team_id]`. Fetch into `$per_list_rows` as array of `['id', 'name', 'date']`.

NOTE: This assumes a `list_columns` join table records which global columns are attached to a list. If the codebase uses a different mechanism (e.g. cells presence), use this query instead:
```sql
SELECT DISTINCT l.id, l.name, l.date
FROM lists l
JOIN cells ce ON ce.list_id = l.id AND ce.player_id = :player_id
JOIN columns c ON c.id = ce.column_id
    AND c.team_id = :team_id AND c.list_id IS NULL AND c.is_active = TRUE
WHERE l.team_id = :team_id
  AND l.visibility IN ('public', 'protected')
ORDER BY l.date DESC NULLS LAST, l.name
```
(Bind `[$player_id, $team_id, $team_id]`.)

Investigate which join table name exists: run `grep -r "list_columns\|lists_columns\|column_list" src/ database/` to confirm the correct table name. Use the correct table. If neither exists and the schema uses cell presence as the only signal, use the cells-based query.

**Query B — cells for this member across all those lists:**
```sql
SELECT ce.list_id, ce.column_id, ce.value
FROM cells ce
JOIN lists l ON l.id = ce.list_id
    AND l.team_id = :team_id AND l.visibility IN ('public', 'protected')
JOIN columns c ON c.id = ce.column_id
    AND c.team_id = :team_id AND c.list_id IS NULL AND c.is_active = TRUE
WHERE ce.player_id = :player_id
```
Bind `[$team_id, $team_id, $player_id]`. Reshape into `$per_list_cells[list_id][column_id] = value`.

**Compute totals:**
Build `$per_list_totals[column_id]`:
- number column: `SUM` of all numeric values across all lists in `$per_list_cells`
- boolean column: `count_true` (cells with value `'1'` or `'true'`) and `total_lists_with_col` (count of rows in `$per_list_rows` that have at least one cell for this column_id in `$per_list_cells`)

For boolean percentage: `$total_lists_with_col` = count of list_ids in `$per_list_cells` that have an entry for this column_id (not counting lists where the member has no cell at all for that column — the percentage is count_true / total_lists_that_have_this_column_for_any_member).

Actually per spec: "total number of lists that have that column attached (not just the ones shown)" — use a separate count query:
```sql
SELECT c.id AS column_id, COUNT(DISTINCT l.id) AS total_lists
FROM columns c
JOIN cells ce ON ce.column_id = c.id
JOIN lists l ON l.id = ce.list_id
    AND l.team_id = :team_id AND l.visibility IN ('public', 'protected')
WHERE c.team_id = :team_id AND c.list_id IS NULL AND c.is_active = TRUE
GROUP BY c.id
```
Fetch into `$col_list_counts[column_id] = total_lists`.

Pass `$per_list_rows`, `$per_list_cells`, `$per_list_totals`, `$col_list_counts` into template via `use`.

```php
<?php if (!empty($per_list_rows)): ?>
<h5 class="mb-3 mt-4">Listenübersicht</h5>
<div class="table-responsive">
    <table class="table table-sm table-bordered align-middle">
        <thead class="table-light">
            <tr>
                <th class="text-nowrap">Liste</th>
                <th class="text-nowrap text-muted small fw-normal">Datum</th>
                <?php foreach ($global_columns as $col): ?>
                    <th class="text-end text-nowrap"><?= e($col['name']) ?></th>
                <?php endforeach; ?>
            </tr>
        </thead>
        <tbody>
            <?php foreach ($per_list_rows as $list_row): ?>
                <tr>
                    <td class="fw-semibold text-nowrap"><?= e($list_row['name']) ?></td>
                    <td class="text-nowrap text-muted small">
                        <?= $list_row['date'] ? date('d.m.Y', strtotime($list_row['date'])) : '—' ?>
                    </td>
                    <?php foreach ($global_columns as $col): ?>
                        <td class="text-end text-nowrap">
                            <?php
                                $cid = (int)$col['id'];
                                $lid = (int)$list_row['id'];
                                $val = $per_list_cells[$lid][$cid] ?? null;
                                if ($val === null) {
                                    echo '<span class="text-muted">—</span>';
                                } elseif ($col['data_type'] === 'boolean') {
                                    echo in_array($val, ['1', 'true'], true) ? '✓' : '✗';
                                } else {
                                    $n = (float)$val;
                                    echo ($n == floor($n)) ? (int)$n : number_format($n, 2, ',', '.');
                                }
                            ?>
                        </td>
                    <?php endforeach; ?>
                </tr>
            <?php endforeach; ?>
        </tbody>
        <tfoot class="fw-bold">
            <tr>
                <td colspan="2">Gesamt</td>
                <?php foreach ($global_columns as $col): ?>
                    <td class="text-end text-nowrap">
                        <?php
                            $cid    = (int)$col['id'];
                            $totals = $per_list_totals[$cid] ?? null;
                            if ($col['data_type'] === 'number') {
                                $n = (float)($totals['sum'] ?? 0);
                                echo ($n == floor($n)) ? (int)$n : number_format($n, 2, ',', '.');
                            } else {
                                $count_true = (int)($totals['count_true'] ?? 0);
                                $total_lists = (int)($col_list_counts[$cid] ?? 0);
                                $pct = $total_lists > 0 ? round($count_true / $total_lists * 100) : 0;
                                echo $count_true . ' <small class="text-muted fw-normal">(' . $pct . '%)</small>';
                            }
                        ?>
                    </td>
                <?php endforeach; ?>
            </tr>
        </tfoot>
    </table>
</div>
<?php endif; ?>
```

Note on boolean display: use `✓` / `✗` for per-cell values in the body rows. In `<tfoot>`, show count + percentage format matching the spec: `3 (75%)`.
